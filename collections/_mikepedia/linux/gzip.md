Comprime e descomprime arquivos `.gz`

### Modo de uso
```
gzip [opção] "nome do arquivo"
```
_obs.: se não for dado nenhuma opção ele comprime o arquivo_

### Opções

**`-d`**  descomprime o arquivo

**`-f`**  força a ação

**`-k`**  mantém o arquivo original
